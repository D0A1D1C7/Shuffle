### Installation

Only you need is Python 2.7

## Getting started
Shuffle 0.1 (Beta) works with Python 2.7 You can run it with:

```sh
$ python shuffle.py -h
```
Command:
  - -g Generate shell
  - -p Setting Password
  - -u Url to connetec
  
To create a webshell run:
```sh
$ python shuffle.py -g webshell.php -p 123
```
